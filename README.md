## exif2html

exif2html is image gallery generator

## Requirements

 * perl module 
   * Image::ExifTool (sudo apt-get install libimage-exiftool-perl)

## install

1. Check out exif2html

  ~~~ sh
  $ git clone git://github.com/shinoburc/exif2html
  ~~~

## Generate image gallery

  ~~~ sh
  $ cd exif2html
  $ ./exif2html.pl

Have fun!
